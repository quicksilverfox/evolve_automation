A fork of a fork with even more random assorted changes.

Changes highlight:

Gameplay:
* Multiple major fixes in hell soldiers management.
* Multiple changes in mech handling. Fixed detection of the best weapon, fixed softlock with build space managament.
* Implemented Eden stationed mech management.
* Better management of lake ship building. An upgrade system to replace suboptimal ships with spare gems.
* An option to disable excess of Andromeda ships instead of parking them in Gorddon.
* For Witch Hunter, improved Suspiciojn handling should more reliably keep it in safe zone.
* Authority management system for Evil universe

API and snippets:
* trigger.toCost function — new API for snippets to check costs of buildings and techs with affordability
* techs exposed similarly to buildings
* Convenient getters/setters for buildings and techs settings
* Lambda-based tooltip additions prototype
* Ignored researches reworked — now a settings-managed list instead of custom array that is safe for snippet management

Also, many minor bugfixes and improvements.
